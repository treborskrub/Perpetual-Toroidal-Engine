# Perpetual-Toroidal-Engine

A minimal, open-source computational twin of the 2025 Aalto continuous time crystal experiment.

Self-addressed toroidal manifold with Pi indexing + natural-gradient flow.

**Run in Colab (one-click):**  
https://colab.research.google.com/github/treborskrub/Perpetual-Toroidal-Engine/blob/main/grokAalto.ipynb

**How to use:**  
1. Click the Colab link above  
2. Click "Runtime" > "Run all"  
3. Watch the 5,000-tick stress-test with stable toroidal pulses and measurable Aalto-style coupling.

This is a public, free, reusable research tool for studying persistent coherence, self-organization, and quantum many-body dynamics.
